# Owl's Nest Pro

A pixel-faithful, mobile-first static website for **Owl's Nest Real Estate** — navy & gold theme, built with Tailwind CSS and vanilla HTML/JS, deployed on Vercel.

**Live:** https://attope-navy.vercel.app

## Pages

| Page | File |
|------|------|
| Home | `index.html` |
| Property Search | `property-search.html` |
| Featured Properties | `featured-properties.html` |
| Buyers | `buyers.html` |
| Sellers | `sellers.html` |
| About | `about.html` |
| Communities & Projects | `communities-projects.html` |
| Neighborhoods | `neighborhoods.html` |
| Contact | `contact.html` |

## Tech

- **Tailwind CSS** (CDN) with custom navy theme
- Static HTML — no framework, no build step required to serve
- Soleil font, real listing data, client-side property-search filters
- Hosted on **Vercel**

## Deploy

```bash
npx vercel --prod
```
